# 🎰 로또 645 데이터 분석 & 예측 시스템

[![Python](https://img.shields.io/badge/Python-3.8+-blue.svg)](https://www.python.org/)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)
[![Platform](https://img.shields.io/badge/Platform-macOS%20%7C%20Windows%20%7C%20Linux-lightgrey.svg)](https://github.com)

**머신러닝과 통계 분석을 활용한 로또 645 번호 추천 및 패턴 분석 시스템**

> ⚠️ **면책조항**: 본 프로젝트는 데이터 분석 및 통계 학습 목적으로 제작되었습니다. 로또는 독립 시행이므로 과거 데이터가 미래 결과를 보장하지 않습니다. 책임있는 게임을 권장합니다.

---

## 📋 주요 기능

### 🔍 데이터 분석
- **기본 통계**: 번호별 출현 빈도, 구간별 분포, 홀짝 분석
- **시계열 분석**: 핫넘버/콜드넘버, 출현 간격, 트렌드 분석
- **패턴 분석**: 번호 조합, 연속 번호, AC값 분석
- **그리드 패턴**: 7x7 복권 용지 위치 기반 공간 분석 ⭐
- **당첨금 분석**: 1등 당첨금 추이, 당첨자 수 상관관계

### 🤖 머신러닝 예측
- **점수 시스템**: 빈도, 트렌드, 부재 기간 기반 점수 계산
- **확률 모델**: 통계적 확률 분포 기반 샘플링
- **패턴 학습**: 구간/홀짝 분포, 연속 번호, 합계 범위 학습

### 🎯 7가지 추천 전략
1. **⭐ 하이브리드**: 모든 전략 통합 (최고 품질)
2. **📊 점수 기반**: 상위 번호 활용
3. **🎲 확률 가중치**: 확률 분포 샘플링
4. **🔄 패턴 기반**: 빈출 패턴 활용
5. **🎨 그리드 패턴**: 복권 용지 위치 기반 ⭐ NEW
6. **🔢 연속 번호**: 연속 쌍 포함
7. **🎰 무작위**: 대조군

### 📊 시각화
- **18개 차트**: 빈도, 히트맵, 트렌드, 상관관계 등
- **복권 용지 이미지**: 604개 회차 이미지 생성

### 🌐 웹 인터페이스
- **Streamlit 앱**: 6개 페이지 인터랙티브 대시보드
- **고정 모드**: 재현 가능한 번호 생성

## 🚀 빠른 시작

### 1. 환경 설정

\`\`\`bash
# 저장소 클론
git clone https://github.com/MyJYP/lotter645_1227.git
cd lotter645_1227

# 가상환경 설정 (자동)
chmod +x setup.sh
./setup.sh
\`\`\`

### 2. 웹 앱 실행

\`\`\`bash
# 웹 앱 실행
./run_web.sh

# 또는 수동 실행
source venv/bin/activate
cd src
streamlit run web_app.py
\`\`\`

**접속**: http://localhost:8501
